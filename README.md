-二、如何加载
1.在Adobe中：
    直接拖入软件窗口即可。
    或者放置在脚本路径再使用它，默认路径是：
    C:\Program Files\Adobe\Adobe Illustrator [Version]\Presets\[Your Language]\Scripts\

2.在affinity中：
    安装支持脚本的版本(例如v3.2)，并启用MCP连接（Ctrl+,打开设置,）
    安装第三方脚本管理软件Script Manager for Afinity
    将.js插件拖入脚本管理软件，之后点击My Script加载插件
